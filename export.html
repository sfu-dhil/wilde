Data Export

To access different sets of bibliographical, match, and network data generated about the news reports, export the UTF-8 CSV files. The first row contains header information. Fields are quoted and separated by commas.

Data last updated on May 8, 2020.

Volume of news
Export all of the metadata for all the reports as
  • Id
  • Date
  • Word count
  • Document matches
  • Paragraph matches
  • Newspaper Title
  • Region
  • City
  • Language

  • Report city

  • Match region
  • Match city
  • Match type
  • Match similarity
