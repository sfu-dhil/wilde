Data Export

To access different sets of bibliographical, match, and network data generated about the news reports, export the UTF-8 CSV files. The first row contains header information. Fields are quoted and separated by commas.

Volume of news
Export all of the metadata for all the reports as
  1. Id
  2. Date
  3. Word count
  4. Document matches
  5. Paragraph matches
  6. Newspaper Title
  7. Region
  8. City
  9. Language
Matching documents
Export all of the document-level matching data as
  1. Source Id
  2. Source date
  3. Source paper
  4. Source region
  5. Target Id
  6. Target date
  7. Target paper
  8. Target region
  9. Match type
  10. Match
Matching paragraphs
Export all of the document-level matching data as
  1. Source Id
  2. Source date
  3. Source paper
  4. Source region
  5. Source city
  6. Source language
  7. Target Id
  8. Target date
  9. Target paper
  10. Target region
  11. Target city
  12. Target language
  13. Match type
  14. Match
Signatures
Export a list of signed reports as
  1. Id
  2. Date
  3. Newspaper Title
  4. Country
  5. City
  6. Language
  7. Signature
Bibliography
Export a list of all reports as
  1. Id
  2. Date
  3. Newspaper Title
  4. Country
  5. City
  6. Language

Gephi Files - Document Level

Node Table
Export a list of all documents that contain matches as
  1. ID
  2. Label
  3. Language
  4. Region
  5. City
Match Table
Export a list of all document-level matches as
  1. Source
  2. Target
  3. Type
  4. Match Type
  5. Weight
